# PHP Daily Task Tracker

A simple PHP project to track daily tasks. Stores tasks in JSON and allows adding/removing tasks.  

## How to run

1. Place folder in your PHP server folder (XAMPP, MAMP, Laragon, etc.)
2. Open `public/index.php` in browser
3. Add tasks and track daily progress

## Why this is good for hiring managers

- Uses PHP OOP → shows structured backend skills
- Handles JSON → shows basic persistence & data handling
- Simple front-end integration → practical real-world functionality
- Small and self-contained → easy to read & understand
